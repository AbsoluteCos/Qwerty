Arrays

Variables in JavaScript can store any single value you want, but what if you want to store multiple values? For that, we use arrays, like this: 

const myArray = ["foo", "bar", "baz"]

If we now want to access the contents of the array, we can do this: 

myArray[0]

This will give us "foo". Note that because element 0 of the array is the first one, element 1 will be the second one, and so on. This is counterintuitive, so be careful when accessing elements of arrays.

All arrays have a set of methods you can use to modify the array using the dot selector (.), one of which includes the .length() function. This function, as you probably can tell, returns how many elements are in the array.
